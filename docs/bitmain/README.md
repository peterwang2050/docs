# BM 系列

BM 系列是瑞莎基于 SOPHGO BM1684x 研发的一系列计算模块和边缘智算盒子。它有高达32TOPS@INT8的算力， 具备强大完备的深度学习工具链、最高达32路视频处理能力。用途广泛，可靠性高。
