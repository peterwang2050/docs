| 6   | GND        |
| 7   | SDMMC_D0   |
| 8   | SDMMC_D1   |
| 9   | SDMMC_DETN |
| 10  | GND        |
| 11  | GND        |
| 12  | GND        |
| 13  | GND        |

### USB 2.0 HOST Type-A

| PIN | Name           |
| --- | -------------- |
| 1   | VCC5V0_USB20   |
| 2   | USB20_HOST1_DM |
| 3   | USB20_HOST1_DP |
| 4   | GND            |

### USB Type-C

| PIN | Name       |
| --- | ---------- |
| 1   | DBG_USB_DP |
| 2   | DBG_USB_DN |
