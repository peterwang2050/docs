# 产品介绍

## 主要特性

Radxa E20C 是一种超小型网络计算机，提供广泛的网络功能。

Radxa E20C 为创客、物联网爱好者、业余爱好者、PC DIY 爱好者及其他人提供了一个可靠且功能强大的平台，将他们的想法转化为现实。
