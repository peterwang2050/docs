## Radxa Model-Zoo 简介

Radxa Model-Zoo 基于 [SOPHONSDK](https://www.sophon.ai/product/introduce/bmnn-sdk.html) 接口进行开发，提供一系列主流算法的移植例程。包括基于 [TPU-MLIR](../../model-compile/tpu_mlir_env) 的模型编译与量化，基于 BMRuntime 的推理引擎移植，以及基于 BMCV/OpenCV 的前后处理算法移植。

[SOPHONSDK](https://www.sophon.ai/product/introduce/bmnn-sdk.html) 是算能科技基于其自主研发的深度学习处理器所定制的深度学习 SDK，涵盖了神经网络推理阶段所需的模型优化、高效运行时支持等能力，为深度学习应用开发和部署提供易用、高效的全栈式解决方案。

Radxa-Model-Zoo 提供的例子从易到难分为 tutorial、sample、application 三个模块，tutorial 模块存放一些基础接口的使用示例，sample 模块存放一些经典算法在 SOPHONSDK 上的串行示例，application 模块存放一些典型场景的典型应用。

- 在 Airbox 上克隆 radxa model-zoo 仓库
  ```bash
  git clone https://github.com/zifeng-radxa/Radxa-Model-Zoo.git
  ```

- 在 X86 工作站上克隆 radxa model-zoo 仓库
  ```bash
  git clone https://github.com/zifeng-radxa/Radxa-Model-Zoo.git
  ```
